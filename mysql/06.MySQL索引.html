зҙўеј•дёәд»Җд№ҲиғҪжҸҗй«ҳ MySQL жҹҘиҜўж•ҲзҺҮпјҹ вҖӢ

зҙўеј•е°ұеғҸдёҖжң¬д№Ұзҡ„зӣ®еҪ•пјҢиғҪи®© MySQL еҝ«йҖҹе®ҡдҪҚж•°жҚ®пјҢйҒҝе…Қе…ЁиЎЁжү«жҸҸгҖӮ

е®ғдёҖиҲ¬жҳҜ B+ ж ‘з»“жһ„пјҢжҹҘжүҫж•ҲзҺҮжҳҜ O(log n)пјҢжҜ”д»ҺеӨҙеҲ°е°ҫжү«дёҖйҒҚж•°жҚ®иҰҒеҝ«еҫ—еӨҡ

йҷӨдәҶжҹҘеҫ—еҝ«пјҢзҙўеј•иҝҳиғҪеҠ йҖҹжҺ’еәҸгҖҒеҲҶз»„гҖҒиҝһжҺҘзӯүж“ҚдҪңгҖӮ

йЎ№зӣ®дёӯжңҖеёёи§Ғзҡ„еҒҡжі•е°ұжҳҜйҖҡиҝҮ create index дёәз»Ҹеёёз”ЁдҪңжҹҘиҜўжқЎд»¶зҡ„еӯ—ж®өе»әзҙў

иғҪз®ҖеҚ•иҜҙдёҖдёӢзҙўеј•зҡ„еҲҶзұ»еҗ—пјҹ вҖӢ

  • д»ҺеҠҹиғҪдёҠеҲҶзұ»зҡ„иҜқпјҢжңүдё»й”®зҙўеј•гҖҒе”ҜдёҖзҙўеј•гҖҒе…Ёж–Үзҙўеј•пјӣ
  • д»Һж•°жҚ®з»“жһ„дёҠеҲҶзұ»зҡ„иҜқпјҢжңү B+ ж ‘зҙўеј•гҖҒе“ҲеёҢзҙўеј•пјӣ
  • д»ҺеӯҳеӮЁеҶ…е®№дёҠеҲҶзұ»зҡ„иҜқпјҢжңүиҒҡз°Үзҙўеј•гҖҒйқһиҒҡз°Үзҙўеј•гҖӮ

дё»й”®зҙўеј•е’Ңжҷ®йҖҡзҙўеј•д»Җд№ҲеҢәеҲ«пјҹ вҖӢ

дё»й”®зҙўеј• вҖӢ

еңЁ InnoDB дёӯпјҢдё»й”®зҙўеј•е°ұжҳҜ иҒҡз°Үзҙўеј•гҖӮ

зү№зӮ№пјҡ

  • иЎЁж•°жҚ®жң¬иә«жҢүдё»й”® B+ ж ‘з»„з»Ү
  • еҸ¶еӯҗиҠӮзӮ№еӯҳзҡ„жҳҜж•ҙиЎҢж•°жҚ®
  • дёҖеј иЎЁеҸӘиғҪжңүдёҖдёӘиҒҡз°Үзҙўеј•
  • дё»й”®дёҚиғҪдёәз©әпјҢдё”е”ҜдёҖ

жҷ®йҖҡзҙўеј• вҖӢ

жҷ®йҖҡзҙўеј•дёҖиҲ¬жҳҜ дәҢзә§зҙўеј•гҖӮ

зү№зӮ№пјҡ

  • еҸ¶еӯҗиҠӮзӮ№еӯҳзҡ„жҳҜзҙўеј•еҲ—еҖј + дё»й”®еҖј
  • жҹҘиҜўеҰӮжһңдёҚиғҪзӣҙжҺҘд»Һзҙўеј•жӢҝеҲ°е…ЁйғЁеӯ—ж®өпјҢе°ұйңҖиҰҒж №жҚ®дё»й”®еҶҚеҺ»иҒҡз°Үзҙўеј•дёӯжҹҘдёҖж¬ЎпјҢиҝҷдёӘиҝҮзЁӢеҸ« еӣһиЎЁ
  • еҸҜд»ҘжңүеӨҡдёӘжҷ®йҖҡзҙўеј•
  • еҸҜд»ҘйҮҚеӨҚпјҢд№ҹеҸҜд»Ҙдёә nullпјҲеҸ–еҶідәҺе…·дҪ“зәҰжқҹпјү

ж ёеҝғеҢәеҲ« вҖӢ

  1. дё»й”®зҙўеј•жҳҜиҒҡз°Үзҙўеј•пјҢжҷ®йҖҡзҙўеј•жҳҜдәҢзә§зҙўеј•
  2. дё»й”®зҙўеј•еҸ¶еӯҗиҠӮзӮ№еӯҳж•ҙиЎҢж•°жҚ®пјҢжҷ®йҖҡзҙўеј•еҸ¶еӯҗиҠӮзӮ№еӯҳдё»й”®еҖј
  3. дё»й”®зҙўеј•еӨ©з„¶е”ҜдёҖдё”дёҚиғҪдёәз©әпјҢжҷ®йҖҡзҙўеј•дёҚдёҖе®ҡ
  4. жҷ®йҖҡзҙўеј•жҹҘиҜўеёёеёёйңҖиҰҒеӣһиЎЁпјҢдё»й”®зҙўеј•зӣҙжҺҘе®ҡдҪҚж•ҙиЎҢ

зҙўеј•е“Әдәӣжғ…еҶөдёӢдјҡеӨұж•Ҳе‘ўпјҹ вҖӢ

жҜ”еҰӮзҙўеј•еҲ—дҪҝз”ЁдәҶеҮҪж•°гҖҒдҪҝз”ЁдәҶйҖҡй…Қз¬ҰејҖеӨҙзҡ„жЁЎзіҠжҹҘиҜўгҖҒиҒ”еҗҲзҙўеј•дёҚж»Ўи¶іжңҖе·ҰеүҚзјҖеҺҹеҲҷпјҢжҲ–иҖ…дҪҝз”Ё or зҡ„ж—¶еҖҷйғЁеҲҶеӯ—ж®өж— зҙўеј•зӯүгҖӮ

зҙўеј•дёҚйҖӮеҗҲе“ӘдәӣеңәжҷҜе‘ўпјҹ вҖӢ

  1. еҢәеҲҶеәҰдҪҺзҡ„еҲ—пјҢеҸҜд»Ҙе’Ңе…¶д»–й«ҳеҢәеҲҶеәҰзҡ„еҲ—з»„жҲҗиҒ”еҗҲзҙўеј•гҖӮ
  2. йў‘з№Ғжӣҙж–°зҡ„еҲ—пјҢзҙўеј•дјҡеўһеҠ жӣҙж–°зҡ„жҲҗжң¬гҖӮ
  3. TEXTгҖҒBLOB зӯүеӨ§еҜ№иұЎзұ»еһӢзҡ„еӯ—ж®өпјҢеҸҜд»ҘдҪҝз”ЁеүҚзјҖзҙўеј•гҖҒе…Ёж–Үзҙўеј•жӣҝд»ЈгҖӮ
  4. еҪ“иЎЁзҡ„ж•°жҚ®йҮҸеҫҲе°Ҹзҡ„ж—¶еҖҷпјҢдёҚи¶…иҝҮ 1000 иЎҢпјҢе…ЁиЎЁжү«жҸҸеҸҜиғҪжҜ”дҪҝз”Ёзҙўеј•жӣҙеҝ«гҖӮ

дёәд»Җд№Ҳ InnoDB иҰҒдҪҝвҪӨ B+ж ‘дҪңдёәзҙўеј•пјҹ вҖӢ

еӣ дёә b+ж ‘жҳҜдёҖз§Қй«ҳеәҰе№іиЎЎзҡ„еӨҡи·ҜжҹҘжүҫж ‘пјҢиғҪеӨҹйҷҚдҪҺзЈҒзӣҳзҡ„иҜ»еҶҷж¬Ўж•°пјҢеҗҢж—¶ж”ҜжҢҒжңүеәҸйҒҚеҺҶе’ҢиҢғеӣҙжҹҘжүҫ

е“ҲеёҢиЎЁдёҚж”ҜжҢҒиҢғеӣҙжҹҘжүҫпјҢдәҢеҸүж ‘еұӮзә§еӨӘж·ұпјҢb ж ‘дёҚеӨӘж–№дҫҝиҢғеӣҙжү«жҸҸ

B+ж ‘зҡ„еҸ¶вјҰиҠӮзӮ№жҳҜеҚ•еҗ‘й“ҫиЎЁиҝҳжҳҜеҸҢеҗ‘й“ҫиЎЁпјҹеҰӮжһңд»ҺвјӨеҖјеҗ‘вј©еҖјжЈҖзҙўпјҢеҰӮдҪ•ж“ҚдҪңпјҹ вҖӢ

B+ж ‘зҡ„еҸ¶вјҰиҠӮзӮ№жҳҜйҖҡиҝҮеҸҢеҗ‘й“ҫиЎЁиҝһжҺҘзҡ„пјҢиҝҷж ·еҸҜд»ҘвҪ…дҫҝиҢғеӣҙжҹҘиҜўе’ҢеҸҚеҗ‘йҒҚеҺҶгҖӮ

еҪ“жү§вҫҸиҢғеӣҙжҹҘиҜўж—¶пјҢеҸҜд»Ҙд»ҺиҢғеӣҙзҡ„ејҖе§ӢзӮ№жҲ–з»“жқҹзӮ№ејҖе§ӢпјҢеҗ‘еүҚжҲ–еҗ‘еҗҺйҒҚеҺҶгҖӮ

еңЁйңҖиҰҒеҜ№ж•°жҚ®иҝӣвҫҸйҖҶеәҸеӨ„зҗҶж—¶пјҢеҸҢеҗ‘й“ҫиЎЁвҫ®еёёжңүвҪӨгҖӮ

еҰӮжһңйңҖиҰҒеңЁ B+ж ‘дёӯд»ҺвјӨеҖјеҗ‘вј©еҖјиҝӣвҫҸжЈҖзҙўпјҢеҸҜд»Ҙе…Ҳе®ҡдҪҚеҲ°жңҖеҸідҫ§иҠӮзӮ№пјҢжүҫеҲ°еҢ…еҗ«жңҖвјӨеҖјзҡ„еҸ¶вјҰиҠӮзӮ№гҖӮд»Һж №иҠӮзӮ№ејҖе§Ӣеҗ‘еҸійҒҚеҺҶж ‘зҡ„вҪ…ејҸе®һзҺ°гҖӮ

зҙўеј•дёәд»Җд№ҲвҪӨ B+ж ‘дёҚвҪӨжҷ®йҖҡвјҶеҸүж ‘пјҹ вҖӢ

еңЁжһҒз«Ҝжғ…еҶөдёӢпјҢжҷ®йҖҡдәҢеҸүж ‘дјҡйҖҖеҢ–жҲҗдёәй“ҫиЎЁпјҢиҝҷж—¶еҖҷж ‘зҡ„й«ҳеәҰе°ұзӯүдәҺж•°жҚ®йҮҸ

жӯӨж—¶жҹҘжүҫ id=7 е°ұйңҖиҰҒ 7 ж¬Ў I/O ж“ҚдҪңпјҢзӣёеҪ“дәҺе…ЁиЎЁжү«жҸҸгҖӮвҪҪ B+ ж ‘дҪңдёәеӨҡеҸүе№іиЎЎж ‘пјҢиғҪе°Ҷж•°дәҝзә§зҡ„ж•°жҚ®йҮҸжҺ§еҲ¶еңЁ 3-4 еұӮзҡ„ж ‘вҫјпјҢиғҪжһҒвјӨеҮҸе°‘зЈҒзӣҳзҡ„ I/O ж¬Ўж•°

дёәд»Җд№ҲдёҚвҪӨе№іиЎЎвјҶеҸүж ‘е‘ўпјҹ вҖӢ

е№іиЎЎвјҶеҸүж ‘иҷҪз„¶и§ЈеҶідәҶжҷ®йҖҡвјҶеҸүж ‘зҡ„йҖҖеҢ–й—®йўҳпјҢдҪҶжҜҸдёӘиҠӮзӮ№жңҖеӨҡеҸӘжңүдёӨдёӘвјҰиҠӮзӮ№зҡ„й—®йўҳдҫқз„¶еӯҳеңЁ

иҒҡж—Ҹзҙўеј•е’Ңвҫ®иҒҡж—Ҹзҙўеј•жңүд»Җд№ҲеҢәеҲ«пјҹ вҖӢ

иҒҡз°Үзҙўеј•зҡ„еҸ¶вјҰиҠӮзӮ№еӯҳеӮЁдәҶе®Ңж•ҙзҡ„ж•°жҚ®вҫҸпјҢж•°жҚ®е’Ңзҙўеј•жҳҜеңЁвјҖиө·зҡ„

иҒҡз°Үзҙўеј•зҡ„еҸ¶вјҰиҠӮзӮ№еҸӘеҢ…еҗ«дәҶдё»й”®еҖјпјҢйңҖиҰҒйҖҡиҝҮеӣһиЎЁжҢүз…§дё»й”®еҺ»иҒҡз°Үзҙўеј•жҹҘжүҫе…¶д»–еҲ—зҡ„еҖјпјҢе”ҜвјҖзҙўеј•гҖҒжҷ®йҖҡзҙўеј•зӯүвҫ®дё»й”®зҙўеј•йғҪжҳҜвҫ®иҒҡз°Үзҙўеј•

еӣһиЎЁдәҶи§Јеҗ—пјҹ вҖӢ

еҪ“дҪҝвҪӨвҫ®иҒҡз°Үзҙўеј•иҝӣвҫҸжҹҘиҜўж—¶пјҢ MySQL йңҖиҰҒе…ҲйҖҡиҝҮвҫ®иҒҡз°Үзҙўеј•жүҫеҲ°дё»й”®еҖјпјҢз„¶еҗҺеҶҚж №жҚ®дё»й”®еҖјеӣһеҲ°иҒҡз°Үзҙўеј•дёӯжҹҘжүҫе®Ңж•ҙж•°жҚ®вҫҸпјҢиҝҷдёӘиҝҮзЁӢз§°дёәеӣһиЎЁ

иҰҶзӣ–зҙўеј•дәҶи§Јеҗ—пјҹ вҖӢ

иҰҶзӣ–зҙўеј•жҢҮзҡ„жҳҜпјҡжҹҘиҜўжүҖйңҖзҡ„еӯ—ж®өе…ЁйғЁйғҪеңЁзҙўеј•дёӯпјҢдёҚйңҖиҰҒеӣһиЎЁпјҢд»Һзҙўеј•в»ҡе°ұиғҪзӣҙжҺҘиҝ”еӣһз»“жһң

еҸҜд»Ҙе°Ҷвҫјйў‘жҹҘиҜўзҡ„еӯ—ж®өпјҲеҰӮ WHERE жқЎд»¶е’Ң SELECT еҲ—пјүз»„еҗҲдёәиҒ”еҗҲзҙўеј•пјҢе®һзҺ°иҰҶзӣ–зҙўеј•гҖӮ

д»Җд№ҲжҳҜжңҖе·ҰеүҚзјҖеҺҹеҲҷпјҹ вҖӢ

жңҖе·ҰеүҚзјҖеҺҹеҲҷжҢҮзҡ„жҳҜпјҡ MySQL дҪҝвҪӨиҒ”еҗҲзҙўеј•ж—¶пјҢеҝ…йЎ»д»ҺжңҖе·Ұиҫ№зҡ„еӯ—ж®өејҖе§ӢеҢ№й…ҚпјҢжүҚиғҪе‘Ҫдёӯзҙўеј•гҖӮ

дёҖдёӘжҖ§еҲ«еӯ—ж®өпјҢиҝҷдёӘеӯ—ж®өдёҖе®ҡдёҚиғҪеҚ•зӢ¬дҪңдёҖдёӘзҙўеј•еҗ— вҖӢ
