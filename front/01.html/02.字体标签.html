文本样式标签 ​

1. 下划线、删除线和斜体 ​

  • <u>：下划线标记
  • <s>或<del>：删除线标记
  • <i>或<em>：斜体标记

实际应用

这些标签常用于小装饰、小图标。例如京东网站中的小图标就常用<i>标签实现。

2. 粗体标签 ​

  • <b>或<strong>（已废弃）

注意

<b>和<font>标签在 HTML5 中已被废弃，建议使用 CSS 样式替代。

3. 字体标签<font>（已废弃） ​

属性：

  • color：设置字体颜色（单词、十六进制或 RGB 值）
  • size：设置字体大小（1-7）
  • face：设置字体类型

示例：

html
<font face="微软雅黑" color="#FF0000" size="10">示例文本</font>

  • <sup>：上标
  • <sub>：下标

记忆技巧：

  • sup：super（上方）
  • sub：sub（下方）
